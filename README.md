# \# Educative Software



\## Authors:

Esteban Alexander Basurto Farias, Odalys Yajaira Chavez Jacome



\## Introduction

This project will create a system designed to help teachers manage students with different learning paces and levels of understanding. It will provide tools to track academic performance, adapt learning activities, and promote participation in the classroom, making the learning process more inclusive and efficient.



\## Functionalities

\-Create accounts for teachers and students

\-Add and remove students from courses

\-Record grades for assignments, exams, and participation

\-View each student’s academic progress

\-Assign tasks with different difficulty levels

\-Conduct interactive classroom activities

\-Allow anonymous responses to encourage participation

\-Generate alerts for underperforming students

\-Upload and manage educational materials

\-Generate course statistics

\-Classify students based on performance levels

\-Create groups according to performance
